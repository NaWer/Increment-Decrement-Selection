# Increment Decrement Selection

Very simple plugin to increment and decrement selection in SublimeText 3


## Default key bindings

```json
[
	{ "keys": ["ctrl+alt+i"], "command": "increment" },
	{ "keys": ["ctrl+alt+k"], "command": "decrement" }
]
```
